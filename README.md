# Ana Sayfa

## Description
This is the homepage of a website. It includes a navigation bar, a base title section with a carousel, and links to other pages such as 'Hakkımızda', 'İletişim', 'Ürünler', and 'Kampanyalar'.

## Features
- Navigation bar for easy access to different sections.
- Carousel for showcasing products or services.
- Links to additional pages like 'Hakkımızda', 'İletişim', 'Ürünler', and 'Kampanyalar'.

## Installation
No installation is required. Simply open the `index.html` file in a web browser.

## Usage
Open the `index.html` file in a web browser to view the homepage.

## Tech Stack
- HTML
- CSS
- Bootstrap 4.6.2
